# Road to Exile

**Road to Exile** is a text-based adventure game in Python. Explore forests, mountains, rivers, and survive as an exile.  

## How to play

1. Make sure Python 3 is installed.
2. Download `road_to_exile.py`.
3. Open your terminal or command prompt.
4. Navigate to the folder containing `road_to_exile.py`.
5. Run the game:
```bash
python road_to_exile.py
